## O que se pode fazer na Página Pessoas?

### Consultas ->

### Realizadas através de três tabelas

![Pessoas](./images/tabelas.png "Pessoas")

{/* TABELA PESSOAS */}

## Tabela Pessoas ->

![Pessoas](./images/table-pessoas.png "Pessoas")

- **Pesquisa:** Permite a Pesquisa através do nome
- **Adionar Pessoa:** Permite adicionar uma nova pessoas
- **Exportar:** Permite exportar os dados para Excel
- **Colunas:** ID, Nome, Título de Nobreza e País
- **Funções possíveis: ->**<br />
  &nbsp;&nbsp;&nbsp;&nbsp; - Editar Dados de uma Pessoa<br />
  &nbsp;&nbsp;&nbsp;&nbsp; - Ao clicar, permite saber mais detalhes da embarcação<br />

### Detalhes da Pessoa ->

![Pessoas](./images/menu-detalhes-pessoa.png "Pessoas")

- **Detalhes:**<br />
  &nbsp;&nbsp;&nbsp;&nbsp; - Nome<br />
  &nbsp;&nbsp;&nbsp;&nbsp; - Título Nobreza associados<br />
  &nbsp;&nbsp;&nbsp;&nbsp; - Tabela de Cargo associados<br />
  &nbsp;&nbsp;&nbsp;&nbsp; - Tabela de Embarcações associados<br />
- **Adicionar Cargo:** Permite associar um Cargo à pessoa pretendida<br />
